# Cybersecurity & CTF Writeups

This repository is a collection of my detailed solutions and walkthroughs for various Capture The Flag (CTF) challenges and cybersecurity platforms. My goal is to document my learning process in reverse engineering, binary exploitation, and general security concepts.

## 🚀 Platforms

### 🚩 picoCTF
* **Intro to Assembly:** Walkthroughs for the Bit-O-Asm series focusing on x86_64 assembly logic, register manipulation, and branching.

### 🛠️ Future Goals (Upcoming)
* **TryHackMe:** Detailed reports on network security and penetration testing rooms.
* **Binary Exploitation:** Deep dives into buffer overflows and shellcoding.

## 📁 Repository Structure
The repository is organized by platform and category to make navigation easy:
* `picoCTF/` — Challenges from the picoCTF platform.
    * `Intro-to-Assembly/` — Assembly-specific challenges.
    * `Intro-to-Debuggers/` — gbd challenges.

* **GDB:** Used for dynamic analysis and debugging binaries.
* **Ghidra:** For static analysis and decompilation.
